# goftx
A Golang implementation of the ![FTX REST API specification](https://docs.ftx.com/#overview).

## Usage

Import the package
```
go get -u github.com/yaustn/goftx
```

Example usage
```
go run example/main.go
```

## Authentication
Create a `secrets.yml` file
